## KEY REMINDERS

1. **Block tables**: First row must be merged and contain the block name (e.g., `Hero (split)`, `Cards (icons)`, `Pricing`)

2. **Section Metadata tables**: First row merged with `Section Metadata`, then key-value pairs below

3. **Horizontal lines**: Insert between sections (Insert > Horizontal line in Google Docs)

4. **Links/Buttons**: Select text, press Cmd+K, paste URL

5. **Images**: Upload tree.svg to your media folder, then insert it in the hero table

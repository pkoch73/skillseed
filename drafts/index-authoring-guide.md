# Quick Reference: Google Docs Actions

1. **Insert table**: Insert > Table
2. **Merge cells**: Select cells > Format > Merge cells
3. **Insert horizontal rule**: Insert > Horizontal line (separates sections)
4. **Create link**: Select text > Cmd+K (Mac) or Ctrl+K (Windows)
5. **Bold text**: Cmd+B / Ctrl+B
6. **Insert image**: Insert > Image > Upload from computer
